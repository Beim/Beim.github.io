maven+SSM-frameworks

Posted on 2017-05-30   |     |   visitors

给出完整项目压缩包和项目结构图
百度网盘下载

步骤

环境

eclipse
MySQL

创建表

在数据库mybatis下创建表user_t

创建maven web项目

选择webapp

GroupId site.beim
ArtifactId HelloSSM


得到项目如下

右键项目，选择build path, 在Order and Export中选择JRE System Library, 点击OK。可以看到之前missing的src/main/java和src/test/java自动创建

右键项目，选择Properties, 选择Project Facets, 在Runtimes中添加Tomcat,点击OK。

在pom.xml中添加依赖 查看 pom.xml

整合Spring与MyBatis

配置文件统一放在src/main/resources目录下
建立jdbc.properties, spring-mybatis.xml, log4j.properties
查看 jdbc.properties
查看 spring-mybatis.xml
查看 log4j.properties

接下来使用MyBatis Generator 自动创建代码 压缩包下载
使用到的jar包：mybatis-3.2.7.jar mybatis-generator-core-1.3.2.jar mysql-connector-java-5.1.25-bin.jar
配置文件：generatorConfig.xml
执行文件：generate.sh

执行generate.sh之后将src目录下的文件拷贝到项目中,如图

新建包site.beim.service, 新建类IUserService.
新建包site.beim.service.impl, 新建类UserServiceImpl.
IUserService.java
UserServiceImpl.java

测试一下整合是否成功，在src/test/java目录下新建包site.beim.testmybatis, 新建类TestMyBatis
TestMyBatis.java

运行TestMyBatis, 如图成功

整合Spring MVC

在src/main/resources目录下新建spring-mvc.xml
spring-mvc.xml

在webapp/WEB-INF目录下配置web.xml
web.xml

测试一下整合是否成功，在src/main/java下建立包site.beim.Controller, 新建类UserController
UserController.java

运行 Run on Server, 访问http://localhost:8080/HelloSSM/user, 如图成功

到此SSM整合完成，可以愉快的开发了
